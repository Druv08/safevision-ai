# AI Model

This folder holds everything related to the YOLO-based PPE detection model.

## Subfolders

- `datasets/` — Raw and processed datasets (images + labels). Will be added later.
- `training/` — Training scripts, `data.yaml`, and training configs.
- `inference/` — Scripts to run model predictions on images, videos, or webcam.
- `outputs/` — Predictions, screenshots, logs, exported weights.
